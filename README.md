# Multimodal UPI Fraud Detection

A comprehensive system to detect, flag, and manage UPI payment fraud featuring a FastAPI backend and a Quasar Vue3 frontend.

## Getting Started
1. Run `bash setup-git.sh` immediately after cloning.
2. Follow the setup instructions printed in the console.
3. Your database and API will be spun up on port `8000`, the frontend on port `9000`.
